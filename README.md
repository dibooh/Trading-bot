# Multi-Strategy Binance Spot Trading Bot

بوت تداول Spot على Binance يجمع 4 استراتيجيات مع إدارة مخاطر صارمة.

## ⚠️ تحذيرات مهمة

- **هذا البوت ليس مضمون الأرباح.** قد تخسر كل رأس مالك.
- اختبر على Testnet أولاً قبل أي شيء.
- لا تستخدم مبلغ لا تستطيع تحمل خسارته.
- الكود مفتوح المصدر، لا توجد ضمانات.

-----

## التثبيت (على Mac)

```bash
# 1. ادخل المجلد
cd binance_bot

# 2. أنشئ بيئة افتراضية
python3 -m venv venv
source venv/bin/activate

# 3. ثبت المتطلبات
pip install -r requirements.txt

# 4. اعمل ملف .env
cp .env.example .env
nano .env   # حط API keys هنا
```

-----

## الحصول على API Keys

### للاختبار الحقيقي (Spot):

1. ادخل <https://www.binance.com/en/my/settings/api-management>
1. أنشئ API key جديد
1. **مهم جداً**: فعّل فقط:
- ✅ `Enable Reading`
- ✅ `Enable Spot & Margin Trading`
- ❌ **لا تفعّل** `Enable Withdrawals` نهائياً
1. حدد IP whitelist (IP جهازك) للأمان

### للاختبار التجريبي (Testnet - مجاني):

1. ادخل <https://testnet.binance.vision/>
1. سجل دخول بحساب GitHub
1. احصل على API keys
1. ضع `USE_TESTNET=true` في `.env`

## كيف يشتغل البوت

### 1. الاستراتيجيات الأربعة

|الاستراتيجية      |الوزن|المنطق                               |
|------------------|-----|-------------------------------------|
|RSI Mean Reversion|30%  |شراء عند oversold، بيع عند overbought|
|MACD Momentum     |25%  |إشارات تقاطع MACD                    |
|Bollinger Bands   |25%  |شراء عند الحد السفلي                 |
|EMA Crossover     |20%  |تأكيد الترند                         |

### 2. القرار النهائي

البوت يجمع الإشارات بالأوزان (-1 إلى +1):

- **شراء** إذا composite ≥ +0.4
- **بيع** إذا composite ≤ -0.4
- **انتظار** بينهما

### 3. إدارة المخاطر (إجبارية)

- ⛔ **Stop Loss**: -2% على كل صفقة
- 🎯 **Take Profit**: +3% على كل صفقة
- 📊 **حجم الصفقة**: 30% من الرصيد كحد أقصى
- 🔢 **عدد الصفقات المفتوحة**: 3 كحد أقصى
- 🛑 **حد الخسارة اليومي**: -5% → البوت يتوقف 24 ساعة
- 🚨 **حد الخسارة الكلي**: -20% → البوت يتوقف نهائياً

### 4. الفحص

كل **5 دقائق** على الشموع **15 دقيقة**.

## استكشاف الأخطاء

### “Missing API_KEY”

تأكد إن ملف `.env` موجود وفيه المفاتيح.

### “Signature error” أو “Invalid API”

- المفاتيح غلط، أو
- ساعة جهازك مو متزامنة (الفرق أكثر من ثانية)

### “NOTIONAL filter” خطأ

الحد الأدنى للصفقة في Binance ~10 USDT. زد الرصيد أو غير `MAX_POSITION_PCT`.

### البوت ما يدخل صفقات

طبيعي - الإشارات تحتاج تجمّع. ممكن يعدي يوم بدون صفقة.

-----

## نصائح من التجربة

1. **خل البوت يشتغل أسبوع كامل** قبل ما تحكم على الأداء
1. **راقب الـ logs يومياً** - شوف القرارات وفهمها
1. **لا تعدل الكود وأنت تخسر** - عاطفي ومدمر
1. **سجل كل تعديل** - حتى تعرف وش يشتغل ووش يخرب

## التعديل والتطوير

كل المتغيرات في `CONFIG` dict في بداية `bot.py`. أهم القيم:

```python
"BUY_THRESHOLD": 0.4,           # كل ما زادت = صفقات أقل بس أدق
"SELL_THRESHOLD": -0.4,
"STOP_LOSS_PCT": 0.02,          # ضيق الستوب
"TAKE_PROFIT_PCT": 0.03,        # وسع الهدف
"STRATEGY_WEIGHTS": {...}       # غير الأوزان حسب اللي يشتغل لك
```
